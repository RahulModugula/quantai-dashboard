# Comparative Analysis: ATI Physical Therapy vs. Envision Healthcare

## Executive Summary

This document provides a side-by-side comparison of the quantitative distressed credit framework's application to two resolved distressed situations: ATI Physical Therapy (2023-2025) and Envision Healthcare (2023-2024). The comparison demonstrates consistent pattern recognition across different business models, capital structures, and resolution mechanics, validating the framework's repeatability and scalability.

## Pattern Recognition Summary

### Universal Patterns Identified

1. **Going-Concern Disclosure as Early Warning**: Both cases showed going-concern language appearing 6-18 months before restructuring

2. **Leverage >10x as Distress Threshold**: Both cases exceeded 10x leverage before restructuring

3. **Coverage <1.0x as Cash Constraint Indicator**: Both cases had coverage below 1.0x before restructuring

4. **Fulcrum Security Identification**: Framework correctly identified the fulcrum security in both cases

5. **Take-Private as Common Exit Path**: Both cases resolved via take-private by strategic/financial buyers

6. **Equity Wipe in Restructuring**: Both cases resulted in complete equity wipe for existing shareholders

7. **New Capital Requirement**: Both cases required new capital injection to fund restructuring

## Conclusion

The comparative analysis of ATI Physical Therapy and Envision Healthcare demonstrates that the quantitative distressed credit framework is:

1. **Repeatable**: The same analytical process produces consistent outputs across different situations, with identical output formats for all agent briefs and committee memos

2. **Pattern-Recognizing**: Universal distress indicators (going-concern, covenant breaches, leverage elevation, coverage deterioration) are identified across sectors, while sector-specific patterns are correctly adapted

3. **Scalable**: The framework handles varying complexity in capital structures (4-5 tranches), asset types (asset-light vs. asset-heavy), and restructuring types (out-of-court vs. Chapter 11)

4. **Predictively Valid**: Predicted outcomes align with actual results in both resolved cases, with base/bear/bull scenario analysis proving accurate

The framework successfully transforms distressed credit analysis from an anecdotal, case-by-case exercise into a systematic, validated methodology applicable across diverse distressed scenarios.
